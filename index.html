Notre engagement envers votre vie privée

التزامنا بخصوصيتك

Chez Budget Tracker, la protection de votre vie privée est notre priorité absolue. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos données lorsque vous utilisez notre application mobile.

في Budget Tracker، تُعدّ حماية خصوصيتك أولويتنا القصوى. توضح سياسة الخصوصية هذه كيفية جمع بياناتك واستخدامها وحمايتها عند استخدام تطبيقنا.

✅ Budget Tracker ne vend jamais vos données personnelles à des tiers.
✅ لا يقوم Budget Tracker ببيع بياناتك الشخصية لأي طرف ثالث.
📊

Données collectées

البيانات التي نجمعها

Données stockées localement sur votre appareil :

البيانات المخزنة محليًا على جهازك:

  • Transactions financières (montant, catégorie, date, description)
  • المعاملات المالية (المبلغ، الفئة، التاريخ، الوصف)
  • Catégories personnalisées créées par l'utilisateur
  • الفئات المخصصة التي أنشأها المستخدم
  • Paramètres de l'application (devise, budget mensuel, préférences)
  • إعدادات التطبيق (العملة، الميزانية الشهرية، التفضيلات)
  • Données de budget planifiées
  • بيانات الميزانية المخططة

Données de diagnostic (anonymes) : En cas de plantage, des rapports techniques anonymisés peuvent être envoyés via Sentry pour nous aider à améliorer la stabilité de l'application. Ces rapports ne contiennent aucune donnée financière personnelle.

بيانات التشخيص (مجهولة المصدر): في حال حدوث عطل، قد تُرسل تقارير تقنية مجهولة المصدر عبر Sentry لمساعدتنا في تحسين استقرار التطبيق. لا تحتوي هذه التقارير على أي بيانات مالية شخصية.

📱

Stockage local uniquement

التخزين المحلي فقط

Toutes vos données financières sont stockées exclusivement sur votre appareil dans une base de données SQLite locale. Aucune donnée financière n'est envoyée sur nos serveurs ni sur internet.

جميع بياناتك المالية مخزنة حصريًا على جهازك في قاعدة بيانات SQLite محلية. لا تُرسل أي بيانات مالية إلى خوادمنا أو عبر الإنترنت.

🗑️ La désinstallation de l'application supprime définitivement toutes vos données.
🗑️ إلغاء تثبيت التطبيق يحذف جميع بياناتك نهائيًا.
🤝

Partage des données

مشاركة البيانات

Nous ne partageons vos données personnelles avec aucun tiers, sauf dans les cas suivants :

لا نشارك بياناتك الشخصية مع أي طرف ثالث، إلا في الحالات التالية:

  • Exportation initiée par l'utilisateur : Vous pouvez exporter vos données en CSV ou JSON. Cette exportation est entièrement sous votre contrôle.
  • التصدير بمبادرة من المستخدم: يمكنك تصدير بياناتك بتنسيق CSV أو JSON. هذا التصدير خاضع تمامًا لسيطرتك.
  • Rapports de plantage (Sentry) : Rapports anonymes d'erreurs techniques, sans données financières.
  • تقارير الأعطال (Sentry): تقارير مجهولة عن الأخطاء التقنية، دون بيانات مالية.
🔐

Autorisations requises

الأذونات المطلوبة

Budget Tracker demande uniquement les autorisations strictement nécessaires :

يطلب Budget Tracker الأذونات الضرورية فقط:

  • Stockage : Pour sauvegarder votre base de données locale et exporter vos fichiers.
  • التخزين: لحفظ قاعدة البيانات المحلية وتصدير ملفاتك.
  • Internet (facultatif) : Uniquement pour l'envoi des rapports de diagnostic anonymes en cas de plantage.
  • الإنترنت (اختياري): فقط لإرسال تقارير التشخيص المجهولة عند حدوث عطل.
🛡️

Sécurité des données

أمان البيانات

Vos données sont protégées par les mécanismes de sécurité intégrés de votre système d'exploitation Android. La base de données SQLite est stockée dans le répertoire privé de l'application, inaccessible aux autres applications sans accès root.

تتمتع بياناتك بحماية آليات الأمان المدمجة في نظام تشغيل Android. قاعدة بيانات SQLite مخزنة في الدليل الخاص بالتطبيق، غير قابلة للوصول من تطبيقات أخرى دون صلاحيات الجذر.

👶

Protection des mineurs

حماية القاصرين

Budget Tracker n'est pas destiné aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles auprès d'enfants.

Budget Tracker غير مخصص للأطفال دون سن 13 عامًا. لا نجمع عن قصد أي بيانات شخصية من الأطفال.
